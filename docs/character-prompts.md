# 16-Bit Big-Head Fighter Character Prompts

These prompts are written for a uniform visual direction:
- super-deformed / big-head cartoon proportions
- 16-bit arcade readability
- simple, bold costume shapes
- sprite-friendly silhouette
- expressive face, chunky hands and feet

## Dual Wield

```text
Create an original 2D fighting game character for a retro 16-bit arcade-style fighting game.

Archetype: dual wield
Combat role: close-range weapons fighter with a fast poke, an overhead or anti-air strike, and a signature special attack
Visual style: super-deformed cartoon fighter, big head style, compact body, exaggerated hands and feet, playful but dangerous, inspired by retro chibi arcade fighters
Rendering style: clean 16-bit pixel art look, bold outlines, flat cel shading, limited color count, readable sprite shapes, retro arcade aesthetic
Proportions: oversized head, short limbs, compact torso, clear weapon silhouettes, expressive face
Pose: full-body neutral fighting stance, side-view friendly, both weapons clearly visible, ready for sprite animation
Background: plain light background, no scenery, no props except the weapons, no text

Design goals:
- readable at low sprite resolution
- weapons must be easy to separate from the body silhouette
- one attack should read clearly as a fast forward poke
- one attack should read clearly as an overhead or anti-air motion
- no tiny costume details
- no photorealism

Character details:
- original character
- personality: sharp, aggressive, stylish
- build: compact and athletic
- weapons: two readable weapons with simple shapes
- colors: 3 to 5 bold colors only
- theme: precision, pressure, dual-weapon flair

Important:
Keep the stance usable as the base for idle, poke, overhead, anti-air, and special attack animation planning.
Big head, small body, strong silhouette, simple readable weapon shapes.
```

## Shared Constraint Add-On

Use this add-on at the end of any prompt if the results become too detailed or too realistic:

```text
Keep the design uniform and simplified.
Prioritize large shapes over detail.
Make it look suitable for 16-bit sprite animation.
Big head, small body, expressive face, chunky hands and feet.
```
